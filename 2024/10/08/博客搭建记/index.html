hexo部署指南

1、一般修改完文章或使用其他功能后，可通过如下指令上传：

hexo g
hexo d

立刻查看静态效果的话可以使用如下指令：

hexo s

2、当部署出现Spawn failed报错时，可以参考下面的方法：

进入站点根目录
cd E:/hexo
 
删除git提交内容文件夹
rm -rf .deploy_git/
 
执行
git config --global core.autocrlf false
 
最后
hexo clean && hexo g && hexo d

3、上传图片

  • 先在_post目录下的images文件夹里面放置图片，然后再 $source$ 目录下的 $images$ 里面继续放置相同图片即可。
  • $markdown$ 里的图片文件命名需要为：/images/id.png，一般直接复制图片的话，需要在开头手动添加 / 才可以。
  • 最后在下图目录下执行三个流程即可上传好图片（这里省略了 $hexo\ d$）

4、常见问题指南

参考了博客：Matery主题新手常见问题，终于解决了代码块的问题！

  • 改代码不生效
    • 如果修改了 _config.yml 文件，调试前一定要执行 hexo clean 命令！
  • 静态网站不能打开，原因是调用 hexo g 的时候没有在 public 中生成 index.html 文件。
    • 解决方法：执行 npm ls --depth 0 命令，可以看出哪些没有安装，将其安装即可（如果看不到的话，可以再开一个文件夹，初始化博客对比看看缺少哪些文件，安装即可，如果不会安装可自行百度。）
  • 删除 node_modules 后想重新创建：
    • 调用 npm install即可重新生成。

配置详细过程：hexo-theme-matery

  • 看完“下载” 和“配置”，再先后执行：$hexo\ g, hexo\ s$，就会发现已经部署成功了。

继续进行一些修改：

  • 前往：themes-hexo-theme-matery-_config.yml中：

    • 找到 $reward$，把打赏的图标给关掉；
    • 找到 $socialLink$，把信息修改为个人的；
    • 找到 $mathjax$，改为 $true$，允许渲染数学公式；
    • 找到 $music$，通过在网易云歌单分享获取那个id，填入即可；
    • 找到 $video$，填入 $b$ 站的iframeUrl：//player.bilibili.com/player.html?isOutside=true&aid=403767234&bvid=BV1AV411L7w7&cid=1207969049&p=1，然后将参数 $height$ 填写为 600，使其视频大小适中；

  • 前往根目录的_config.yml中：

    • 在# Site 中修改成个人信息；

  • 修改两处图片：

    • D:\Blog\themes\hexo-theme-matery\source\favicon.png
    • D:\Blog\themes\hexo-theme-matery\source\medias\logo.png
