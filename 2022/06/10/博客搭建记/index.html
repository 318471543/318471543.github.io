  • 大于号：\ge

hexo部署指南

1、一般修改完文章或使用其他功能后，可通过如下指令上传：

hexo g
hexo d

立刻查看静态效果的话可以使用如下指令：

hexo s

2、当部署出现Spawn failed报错时，可以参考下面的方法：

进入站点根目录
cd E:/hexo
 
删除git提交内容文件夹
rm -rf .deploy_git/
 
执行
git config --global core.autocrlf false
 
最后
hexo clean && hexo g && hexo d

3、上传图片

  • 先在_post目录下的images文件夹里面放置图片，然后再 $source$ 目录下的 $images$ 里面继续放置相同图片即可。
  • $markdown$ 里的图片文件命名需要为：/images/id.png，一般直接复制图片的话，需要在开头手动添加 / 才可以。
  • 最后在下图目录下执行三个流程即可上传好图片（这里省略了 $hexo\ d$）

4、常见问题指南

参考了博客：Matery主题新手常见问题，终于解决了代码块的问题！

  • 改代码不生效
    • 如果修改了 _config.yml 文件，调试前一定要执行 hexo clean 命令！
  • 静态网站不能打开，原因是调用 hexo g 的时候没有在 public 中生成 index.html 文件。
    • 解决方法：执行 npm ls --depth 0 命令，可以看出哪些没有安装，将其安装即可（如果看不到的话，可以再开一个文件夹，初始化博客对比看看缺少哪些文件，安装即可，如果不会安装可自行百度。）
  • 删除 node_modules 后想重新创建：
    • 调用 npm install即可重新生成。

在这里写个每日感受！

2023 年 7 月 20 日

​ 这是一个很特殊的一天，因为大早上下着雨，实验室的人目前只有1位同学到了，其他人都去的很晚，而我仍早起骑着电动车去实验室，当时一手撑伞一手开车，还差一个拐弯就到 9 栋了，结果就在这一刻我摔倒了。腿上有一块贴地被擦伤了（我认为只是擦伤了），然后还在流血。我瞬间不知所措，不知道该干啥了，就先回实验室拿纸止住血。我其实是不想去医院的，因为我从小到大都没自己独立去过医院，平常都是我妈带我去的，我也啥都不知道就坐在那，然后就我妈搞好一切，就直接就诊了，我从不知道流程是什么。好几年都没正式去过医院了，平常打个吊水也是我妈带我去个小诊所完成的，上大学除了刚开学体检去过一次校医院，之后就再也没去过了，因为我内心是能不去医院的就尽量不去。然后我就想到之前骑车摔倒，我妈都带我去医院包扎，然后我内心有个声音，告诉我现在必须去医院，不能再有任何推脱了！上午所有的活动（学习之类的）全部取消，先把伤口处理好！然后我就再骑着小绿去校门口，准备前往这个“令我向往”的校医院。因为刚骑车摔过，虽然再次骑车有心理阴影，但毕竟还是需要电动车的，只不过这次打伞骑车一定要格外小心！（虽然这样是错误的，但目前只能这样快速到达医院了。）到达医院后，我真的是一点流程也不知道，在一楼看到前台没有人，感觉很离谱，然后看了下标志才发现要去二楼。然后我看到终于有人了，就排队等他们弄好。轮到我的时候，我问那个医生我腿摔倒受伤了该怎么办，然后她就说有没有带什么报销啥的，我没听懂，就说啥也没带，校园卡也没带。然后她就问了我学号，然后我就给她了，她想我指了下内科让我找那里的医生，问了内科的医生后他就给我开了个啥我也没懂，就又回到前台找那个医生，付了钱后又又找内科的医生，然后他说，还找我干啥，找护士。奥奥，懂了，原来我要去找护士。然后护士就开始给我处理伤口，我还问她这个严重吗？我感觉我能走。她却说我的伤口很深，要每天来医院上药。我好害怕，我真的感觉我能走，应该没啥事呜呜呜。然后我把这事告诉我妈，我妈说让我再问问要不要打破伤风针，我就去问了那个内科的医生。但他表示医院里没有这个针什么的，也说不需要打，基本上99% 的都不会感染，还问了我摔倒的地方有没有铁钉之类的，我说没有。然后就这样，我算是第一次独自去医院吧，我一般都自认为身体可以，骑车也很稳，基本跟医院无缘，但没想到，我还是骑车不稳，不该下雨天骑车撑伞的。这让我感觉这一天，虽然看起来好痛苦、倒霉，但却是具有象征性的一天，因为我竟然会去医院了。然后我就突然想到记录下每日的感受，感觉都挺有意义的，就放在这个博客里吧！希望我的腿伤没事，希望我可以尽快恢复。

2023 年 7 月 21 日

​ 今天中午吃饭，吃了块鱼，结果竟然卡到刺了，本以为咽下去了，谁知回到实验室开始打比赛的时候又突然感觉到了喉咙中有鱼刺的感觉。一直都无法把鱼刺给“吸上来”，然后就不得不去卫生间在水龙头那里洗，当时一直没搞出来，越弄越恶心，然后就哕出来了，可能吐的比较多的缘故吧，鱼刺没有感觉了，估计是跟着中午吃的饭也一起吐出来了。哎，这两天怎么这么倒霉，两个我最擅长的东西，我全都翻车了呜呜呜。
